#Customer 360

## Project
Customer 360 - UI application that uses couchbase sync service to provide product recommendation.

## Prerequisites
- JDK 1.8 or later
- Maven 3 or later
- MySQL 5.6 or later

## Stack
- Spring MVC
- Spring Security
- Spring Data JPA
- Maven
- JSP
- MySQL

## Run
```mvn jetty:run```
